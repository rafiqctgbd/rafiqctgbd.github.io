Project README

Medicine & Health Reminder is an Android application designed to help users manage medicine schedules, health reminders, and personal medical records securely and conveniently.

Features

  • Medicine reminder alarms
  • Health tracking and notes
  • Medical record management
  • Dose history tracking (taken, snoozed, missed)
  • Pause all reminders temporarily
  • Share medical records via WhatsApp or email
  • Google Drive backup and restore
  • Secure local data storage
  • Notification reminders
  • Optional Google Sign-In support

Data Privacy

User privacy is a core priority of this application.

  • Data is stored locally on the user's own device by default
  • No personal data is stored on developer-owned servers
  • Google Drive backup is completely optional
  • Backup data is stored only in the user's own private Google Drive AppData folder
  • User data is never sold or shared with third parties
  • No advertisements are shown and no advertising SDKs are used

Google Drive Backup

The application optionally supports Google Drive backup and restore using the Google Drive drive.appdata scope. This allows users to securely backup and restore medicine reminders and health records across devices while keeping the data private and isolated from normal Google Drive files.

The App cannot access general Google Drive files, photos, videos, or documents. Only the user's own AppData folder is used.

Permissions Used

  • Internet access — backup and restore via Google Drive
  • Notifications and alarms — medicine reminder alerts
  • Camera and media/storage access — medical image attachments
  • Boot completed — automatically restore reminders after device restart
  • Exact alarm scheduling — ensure reminders fire at precise times
